# SO_Lista02.01
## Controle de Redes

Este desafio consiste em criar uma aplicação, em java, que faça a leitura do IPV4 e realize um teste de Ping com 10 interações.

```
Requesitos:

1. Arquitetura MVC;
2. Os métodos de acesso ao terminal devem ser privados;
3. A aplicação deve ter um menu para selecionar o método desejado.
```
